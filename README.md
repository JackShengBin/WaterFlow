# WaterFlow(瀑布流)
基于collectionView的常用的布局方式
重写UICollectionViewLayout的方法

/**
初始化
*/
prepareLayout

/**
决定cell的排列
*/
layoutAttributesForElementsInRect

/**
返回indePath位置cell的布局属性
*/
layoutAttributesForItemAtIndexPath

/**
返回view的contentSize
*/
collectionViewContentSize
